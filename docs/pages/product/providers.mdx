## Signers
- Manage dVault keys using **secure TSS protocols**.
- Enforce custom vault rules (repayment, liquidation, migration, escape hatch).
- Participate in liquidation triggers and recovery mechanisms.

## Validators
- Run Surge‚Äôs **EVM-compatible execution layer** (`reth` + `CometBFT`).
- Participate in block production, ensuring deterministic finality.
- Validate vault state transitions, liquidation events, and cross-chain execution.

## Bootstrapping the Network
- Surge launches with a **permissioned set of 50+ infra providers**.
- Gradual rollout towards **permissionless participation** via staking + reputation.
- Rewards distributed in **BTC and stablecoins**, aligning incentives across stakeholders.
