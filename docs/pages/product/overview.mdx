Surge is fundamentally a decentralized lending marketplace, where Bitcoiners, stablecoin holders, and infrastructure providers collaborate to enable transparent, on-chain lending - without intermediaries.

The Surge product suite comprises three core interfaces:

- **Borrow Interface** for Bitcoiners seeking liquidity against their BTC
- **LPs Interface** for stablecoin holders and LPs looking to earn yield
- **Infra Provider Tools** for validators, signers, and network operators powering the decentralized backend

At the core of all this is dVaults, decentralized, programmable Bitcoin vaults built using Tapscript and secured via lin22, a threshold schnorr signature protocol. 
These vaults ensure Bitcoin stays verifiably on-chain and unrehypothecated.

:::info[Deterministic Liquidation]
Each dVault encodes deterministic spend paths (Repay, Liquidate, Escape) and a pre-committed PSBT template. If a borrower defaults, liquidation follows these pre-defined, verifiable conditions.  
[Learn more ‚Üí](/tech/dvaults-liquidation)
:::
