# Alpha Testnet Explorer

The [Surge Alpha Testnet Explorer](https://explorer-alphatestnet.surge.dev/) is the official explorer for the Surge Network's alpha testnet. It allows you to:

- View all transactions on the Surge Network
- Track proof submissions and their status
- Monitor block information and network activity
- Search for specific transactions, blocks, or addresses
- Verify proof settlements and their progression to Bitcoin

Access the explorer at: [https://explorer-alphatestnet.surge.dev/](https://explorer-alphatestnet.surge.dev/)
