## Steps to Connect

### 1. Add Surge Alpha Testnet Chain

<img
  src="/assets/add_surge_network.png"
  alt="Add Network Confirmation"
  className="max-w-[60%]shadow-lg my-4"
/>

1. Visit [chains.keplr.app](https://chains.keplr.app)
2. Search for "**Surge Alpha Testnet**"
3. Click "Add to Keplr"
4. Click "Approve"

<img
  src="/assets/add_network_confirmation.png"
  alt="Add Network Confirmation"
  className="max-w-[250px] shadow-lg my-4"
/>

### 2. Create a New Wallet (Skip if you have one)

1. Open Keplr extension
2. Click "Create a new wallet"
3. Follow the setup process

<img
  src="/assets/create_keplr_wallet.png"
  alt="Create Keplr Wallet"
  className="max-w-[250px] shadow-lg my-4"
/>

### 3. Enable Chain Visibility

1. Click the hamburger menu (â˜°) in Keplr
2. Select "Manage chain visibility"
3. Search for "Surge Alpha Testnet"
4. Enable and save changes

<img
  src="/assets/manage_chain_visibilty.png"
  alt="Manage Chain Visibility"
  className="max-w-[250px] shadow-lg my-4"
/>

### 4. Verify Connection

You should now see Surge Alpha Testnet in your wallet list:

<img
  src="/assets/surge_chain_wallet.png"
  alt="Surge Chain Wallet"
  className="max-w-[250px] shadow-lg my-4"
/>

### 5. Get Test Tokens

To get testnet SURG tokens for development, visit our [faucet guide](/developer-guides/wallets-tokens/get-tokens).
