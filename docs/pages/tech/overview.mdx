## Control Plane (Bitcoin)

- BTC collateral is locked in Taproot **dVaults** (Merkelized Abstract Syntax Tree / MAST - based programs).
- BTC never leaves Bitcoin and can move only via **deterministic spend conditions**:
  - **Repayment** requires borrower and signer signatures
  - **Liquidation** (if CR < threshold)
  - **Time-locked** (`OP_CHECKLOCKTIMEVERIFY`) for the loan duration
- A BTC/USD **oracle** drives CR checks and liquidation triggers.
- **Signer layer security**, all BTC spends require quorum approval from MPC signers using <a href="https://github.com/coinbase/cb-mpc" target="_blank" rel="noopener noreferrer" class="opacity-80 hover:opacity-100 hover:underline cursor-pointer transition-colors">Coinbaseâ€™s cb-mpc</a> framework.

## Execution Plane

- Fully EVM compatible, built with reth and CometBFT for deterministic state
- Surge **deploys to existing chains**, recording vault states, liabilities, stability pool balances, and signer epochs
- State transitions on the Surge plane mirror BTC movements, ensuring 1:1 consistency between Bitcoin and stablecoin liabilities

- The initial signer set is permissioned, with a roadmap toward **permissionless** enrollment.
